string_shuffle
==============

Returns a given string with charactars shuffled.

    s = string_shuffle("1234567890");   //  "6059241837"
    s = string_shuffle("12345");        //  "52431"
    s = string_shuffle("1");            //  "1"
    s = string_shuffle("");             //  ""
