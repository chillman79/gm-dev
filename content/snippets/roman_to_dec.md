roman_to_dec
============

Converts a string of Roman numerals to a decimal value.

    dec = roman_to_dec("XLII");     //  dec == 42
    dec = roman_to_dec("MCMXCIX");  //  dec == 1999
    dec = roman_to_dec("MMDXXV");   //  dec == 2525
