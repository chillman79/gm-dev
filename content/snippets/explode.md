explode
=======

Returns an array of strings parsed from a given
string of elements separated by a delimiter.

    names = "Juliett,Victor,Mike,Charlie,Romeo,Oscar";

    array = explode(",", names);

    //  array[0] == "Juliett"
    //  array[1] == "Victor"
    //  array[2] == "Mike"
    //  array[3] == "Charlie"
    //  array[4] == "Romeo"
    //  array[5] == "Oscar"
