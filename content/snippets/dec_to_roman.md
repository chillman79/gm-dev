dec_to_roman
============

Converts a decimal value to a string of Roman numerals.

    roman = dec_to_roman(42);       //  roman == "XLII"
    roman = dec_to_roman(1999);     //  roman == "MCMXCIX"
    roman = dec_to_roman(2525);     //  roman == "MMDXXV"
