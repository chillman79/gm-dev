string_ucfirst
==============

Returns a string with the first character capitalized.

    s = string_ucfirst("hello, world!");    //  "Hello, world!"
    s = string_ucfirst("HELLO, WORLD!");    //  "HELLO, WORLD!"
    s = string_ucfirst("12345");            //  "12345"
    s = string_ucfirst("");                 //  ""
