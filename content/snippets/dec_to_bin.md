dec_to_bin
==========

Converts a decimal value to a string of binary digits.

    bin = dec_to_bin(123);          //  bin == "1111011"
    bin = dec_to_bin(456);          //  bin == "111001000"
    bin = dec_to_bin(789, 16);      //  bin == "0000001100010101"
    bin = dec_to_bin(-43, 16);      //  bin == "1111111111010101"
