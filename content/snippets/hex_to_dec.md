hex_to_dec
==========

Converts a string of hexidecimal digits to a decimal value.

    dec = hex_to_dec("7B");         //  dec == 123
    dec = hex_to_dec("1C8");        //  dec == 456
    dec = hex_to_dec("315");        //  dec == 789
