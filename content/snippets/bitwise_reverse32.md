bitwise_reverse32
=================

Returns the given 32-bit number with bits in reverse order.

    bits = 123456789;               //   123456789 [ 00000111010110111100110100010101 ]
    rev = bitwise_reverse32(bits);  //  2830359264 [ 10101000101100111101101011100000 ]
