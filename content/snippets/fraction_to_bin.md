fraction_to_bin
===============

Converts a fractional value to a string of binary digits.

    bin = fraction_to_bin(0.12500, 5);  //  bin == "00100"
    bin = fraction_to_bin(0.65625, 5);  //  bin == "10101"
    bin = fraction_to_bin(0.96875, 5);  //  bin == "11111"
