factor
======

Returns an array containing the prime factors of a given integer.

    f = factor(   1);   //  [ ]
    f = factor(   5);   //  [ 5 ]
    f = factor(  12);   //  [ 2, 2, 3 ]
    f = factor( 222);   //  [ 2, 3, 37 ]
    f = factor( 256);   //  [ 2, 2, 2, 2, 2, 2, 2, 2 ]
    f = factor(2222);   //  [ 2, 11, 101 ]
