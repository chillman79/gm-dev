bin_to_dec
==========

Converts a string of binary digits to a decimal value.

    dec = bin_to_dec("00100");      //  dec == 4
    dec = bin_to_dec("10101");      //  dec == 21
    dec = bin_to_dec("11111");      //  dec == 31
